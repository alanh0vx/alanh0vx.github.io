# 象棋 — 中國象棋

純客戶端中國象棋遊戲，人機對弈，支援新手教學、存檔讀取等功能。

## 功能

- **人機對弈** — 三種難度（初級 / 中級 / 高級），AI 使用 Alpha-Beta 搜索演算法搭配空步裁剪、延遲走步縮減、吸氣窗口等進階技術
- **新手教學** — 6 課互動教學，涵蓋棋盤、棋子走法、特殊規則、基本殺法、開局及記譜法
- **傳統記譜** — 炮二平五格式的走棋記錄，支援前/後區分，紅黑方顏色標示
- **存檔讀取** — 5 個手動存檔位，離開時提示儲存，支援刪除存檔，所有資料儲存於 localStorage
- **悔棋 / 提示** — 開局前可設定是否允許，每局最多各 3 次
- **託管模式** — 讓電腦代你下完整局棋，隨時可取回控制
- **玩家檔案** — 自訂暱稱及頭像（上傳圖片自動轉為像素風格）
- **戰績統計** — 記錄勝、負、和

## 使用方法

直接用瀏覽器開啟 `index.html` 即可遊玩，無需伺服器、無需安裝、無需網絡連線。

```
支援瀏覽器：Chrome / Firefox / Edge / Safari（最新兩個版本）
```

## 操作說明

| 操作 | 方法 |
|------|------|
| 選擇棋子 | 點擊己方棋子 |
| 移動棋子 | 點擊綠色提示點 |
| 取消選擇 | 點擊空白位置 |
| 查看歷史局面 | 點擊右側走棋記錄 |

## 專案結構

```
index.html          入口頁面
css/style.css       樣式表
js/
  game.js           棋盤表示、FEN 解析、遊戲狀態
  rules.js          走法生成、將軍/將死判定
  board.js          Canvas 棋盤繪製及互動
  ai.js             AI 引擎（Alpha-Beta + 迭代加深）
  notation.js       傳統中文記譜法
  history.js        走棋歷史面板
  storage.js        localStorage 存讀檔
  profile.js        玩家檔案及像素頭像處理
  endgame.js        殘局挑戰模組（暫時停用）
  tutorial.js       新手教學模組
  audio.js          音效（Web Audio API 合成）
  ui.js             對話框、提示訊息
  main.js           主控制器及路由
data/
  endgames.json     殘局資料（暫時停用，待修正後重新啟用）
  openings.json     開局庫
```

## 技術

- 純 HTML5 + CSS3 + JavaScript，無任何框架或依賴
- Canvas 繪製棋盤及棋子
- Web Audio API 合成音效
- localStorage 持久化所有資料
- 從 `file://` 直接開啟即可運行，零網絡請求
